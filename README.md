### 🚀 O que você encontra aqui?
Este repositório centraliza ferramentas desenvolvidas para resolver problemas específicos e explorar diferentes bibliotecas do ecossistema Python:

* **🛡️ Security & Pentesting:** Scripts focados em testes de vulnerabilidade leve (FTP), scanners de rede e ferramentas de automação para análise de segurança.
* **🎲 RPG Systems:** Motores de lógica para jogos de RPG, incluindo geradores de ficha, sistemas de dados e gerenciamento de turnos.
* **⚙️ Automações Variadas:** Scripts para otimização de tarefas cotidianas e manipulação de arquivos.

---

### 🛠️ Diferenciais Técnicos
* **Modularidade:** Código escrito de forma que cada ferramenta possa ser integrada a outros sistemas maiores.
* **Tratamento de Exceções:** Foco na robustez dos scripts para evitar interrupções durante execuções críticas (especialmente em ferramentas de rede).
* **Eficiência:** Uso de bibliotecas nativas e externas (`socket`, `requests`, `random`, etc.) para máxima performance.

---

### ✨ Precisa de uma ferramenta personalizada?
Desenvolvo scripts sob medida para sua necessidade, seja uma automação empresarial, uma ferramenta de segurança ou um sistema de jogo exclusivo.
